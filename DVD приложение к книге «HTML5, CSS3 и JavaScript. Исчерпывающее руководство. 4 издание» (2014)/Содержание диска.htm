AceFTP

Мощный по своим возможностям и бесплатный FTP-клиент.

Adobe Reader

Бесплатная программа, выпускаемая корпорацией Adobe для просмотра электронных публикаций в формате PDF. Требуется для просмотра дополнительных материалов к книге.

Chrome

Google Chrome - веб-браузер производства Google, сочетающий поддержку новейших веб-приложения и удобный поиск с высокой скоростью и стабильностью.

CuteFTP

FTP-клиент, позволяющий безопасно и надежно обмениваться файлами между локальным компьютером и серверами в Интернете.

Cygwin

UNIX-подобная среда и интерфейс командной строки для Microsoft Windows.

FileZilla

Бесплатный мультиязычный FTP-клиент. Поддерживает FTP, SFTP, и FTPS (FTP через SSL/TLS).

Firefox

Быстро работающий браузер, отлично настраиваемый и расширяемый с помощью плагинов.

Flash Player

Медиаплеер, проигрывающий Flash-ролики. Отдельно дистрибутивы для браузера Internet Explorer и для браузеров Mozilla, Firefox, Safari, Opera.

GIFmation

Инструмент для создания анимированных файлов формата  GIF.

GIMP

Бесплатный растровый графический редактор, программа для создания и обработки растровой графики. Частично поддерживается векторная графика. Прилагается руководство на русском языке.

Internet Explorer

Браузер Internet Explorer. Различные версии для разных операционных систем. Для некоторых версий при установке потребуется подключение к Интернету.

jEdit

Кроссплатформенный редактор с открытым исходным кодом, написанный на языке Java. Прилагаемая версия требует установки виртуальной машины Java версии 1.5.0 или выше (дистрибутив также доступен в папке jEdit.

JPEG Cruncher

Инструмент для оптимизации файлов формата  JPG.

MapEdit

Редактор карт изображений для веб-сайтов.

Microsoft Expression Encoder

Инструмент кодирования видео корпорации Microsoft.

Nvu

WYSIWYG HTML-редактор, основанный на компоненте Composer пакета Mozilla Application Suite.

Opera

Компактный и в то же время мощный пакет коммуникационных программ с русским интерфейсом, состоящий из быстрого браузера, почтовой программы и модуля для получения новостей с веб-сайтов в формате RSS.

ProJPEG

Инструмент для оптимизации файлов формата  JPG.

PuTTY

Свободно распространяемый клиент для различных протоколов удалённого доступа, включая SSH, Telnet, rlogin.

QuickTime

Проприетарная технология компании Apple, разработанная для воспроизведения цифрового видео, звука, текста, анимации, музыки и панорамных изображений в различных форматах.

Safari

Веб-браузер корпорации Apple, предназначенный для использования в Windows. Версии с мультимедийным проигрывателем QuickTime и без.

TextPad

Текстовый редактор, предназначенный для работы с текстовыми файлами любых размеров.
